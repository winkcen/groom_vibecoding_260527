# 📟 Y2K 뽀모도로 타이머 (Retro Focus Device)

> 2000년대 초반의 투명 가전(iMac, 게임보이), 메탈릭 PDA 기기 및 다마고치 감성을 듬뿍 담은 **Y2K 스타일 뽀모도로 타이머**입니다.
> 삐삑- 아날로그 비프음 피드백과 함께 자신만의 픽셀 스티커로 기기를 꾸미며 집중력을 극대화할 수 있는 교육용 MVP 웹앱입니다.

---

## 💡 입문자 디자인 & 기능 포인트

1.  **3D 아날로그 스키어모피즘(Skeuomorphism)**
    *   플랫 디자인에서 벗어나 실제 하드웨어 기기 버튼을 모방한 `outset`/`inset` 테두리 보더 입체감을 연출합니다.
    *   버튼을 누를 때 1px 단위로 가라앉는 `:active` 애니메이션 효과를 구현하여 조작하는 ‘손맛’을 높였습니다.
2.  **Web Audio API를 통한 레트로 사운드 합성**
    *   MP3 파일 연결 없이 브라우저 내장 오디오 신시사이징 코드를 활용해 레트로 전자기기 특유의 "삑-" 소리와 카운트다운 완료 멜로디를 재생합니다.
3.  **다이어리 꾸미기 (다꾸) 스티커 기능**
    *   HTML5 Drag and Drop API를 실습합니다. 스티커를 마우스나 터치로 끌어 기기 프레임 위 원하는 곳에 올리면 상대적인 위치값(`left: X%`, `top: Y%`)으로 정확히 고정 부착됩니다.
    *   부착된 스티커들은 개별 삭제가 가능하며 브라우저 `LocalStorage`에 저장되어 새로고침 후에도 그대로 보존됩니다.
4.  **3가지 Y2K 스타일 디바이스 스킨**
    *   `Cyber Silver` (메탈릭 실버 PDA)
    *   `Translucent Blue` (90년대 말 아이맥/게임보이 스타일 반투명 블루 플라스틱)
    *   `Retro Pink` (2000년대 초반 체리 폴더폰)

---

## 🛠 기능 명세

*   **집중(Focus) 모드 & 휴식(Break) 모드**: 25분 집중과 5분 휴식을 순환 제어하며 LCD 화면이 네온 컬러별로 자동 전환됩니다.
*   **LCD 도트 매트릭스 디스플레이**: 격자무늬 스크린과 깜빡이는 콜론(`:`), 하단에서 부드럽게 흐르는 텍스트 애니메이션을 선사합니다.
*   **사운드 오디오**: START/STOP/RESET 및 스킨 클릭 시 각각 피치가 다른 전자 비프음이 연동됩니다.

## 📂 파일 구조

```text
y2k-pomodoro/
├── index.html     # Win98 스타일 창 및 다마고치 기기 레이아웃 구성
├── style.css      # 3D 베벨 입체 버튼, 3개 스킨 변수 정의, 네온 백라이트 효과
├── app.js         # 타이머 제어, 8비트 사운드 신시사이저, 스티커 드래그 좌표 계산
└── README.md      # 본 문서 (프로젝트 안내서)
```

## 🚀 실행 방법

1. `y2k-pomodoro` 디렉토리 내의 **`index.html`** 파일을 더블클릭하여 브라우저에서 즉시 구동합니다.
2. 상단의 스킨 버튼들을 눌러 기기 색상을 변경하고, 우측의 픽셀 스티커들을 기기 위에 드래그하여 나만의 뽀모도로 기기를 커스텀해보세요!
