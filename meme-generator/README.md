# ⚡ 즉석 밈 생성기 (CSS Position Studio)

> 입문자가 웹 개발의 레이아웃 배치 개념 중 가장 헷갈리기 쉬운 **CSS `position` (`relative`와 `absolute`)** 개념을 재미있는 밈(Meme) 제작 실습을 통해 마스터할 수 있도록 설계된 MVP 토이프로젝트입니다.

---

## 💡 입문자 만족도 & 학습 포인트

1.  **CSS Position의 이해 (`relative` & `absolute`)**
    *   부모 요소(`position: relative`)와 그 안에 붕 떠서 돌아다니는 자식 요소(`position: absolute`)의 관계를 완벽히 실습합니다.
    *   글자를 마우스나 손가락으로 드래그할 때 `top`, `bottom`, `left`의 인라인 CSS 속성값(%)이 어떻게 실시간으로 변하는지 시각적으로 직접 목격할 수 있습니다.
2.  **실시간 CSS Inspector (코드 뷰어)**
    *   드래그 위치나 폰트 크기 변경에 맞춰 실시간으로 조립되는 CSS 코드를 보며, 실무 웹디자인 에디터(Figma 등)의 작동 메커니즘을 경험해 봅니다.
3.  **즉각적인 재미와 공유성**
    *   `html2canvas` 라이브러리를 연동하여, 순수 CSS로 화면상에 얹어놓은 텍스트 요소를 그대로 캡처해 고화질 파일(`.png`)로 즉시 소장하고 친구들에게 공유할 수 있습니다.

---

## 🛠 주요 기능 목록

*   **10종 이상의 인기 밈 템플릿**: 랜덤 버튼 하나로 대중적인 밈 이미지와 위트 있는 어구 조합을 즉석 제공합니다.
*   **드래그 앤 드롭 자유 배치**: 이미지 위 텍스트를 원하는 장소에 끌어다 놓으면 가로폭 대비 중앙 정렬(`transform: translateX(-50%)`) 규칙을 지킨 채 좌표가 재정의됩니다.
*   **세부 타이포그래피 설정**: 폰트 크기, Y축 수직 오프셋 비율(%), 폰트 컬러를 슬라이더와 피커로 정밀 튜닝합니다.
*   **커스텀 짤방 제작**: 제공 템플릿 외에도 사용자 PC의 이미지를 가져와 즉석에서 나만의 밈을 구축할 수 있습니다.
*   **고화질 다운로드**: 화면 캡처 중 드래그용 파란 가이드 점선은 자동으로 제외하고 깔끔한 최종 결과물만 내보냅니다.

## 📂 파일 구조

```text
meme-generator/
├── index.html     # 에디터 UI 마크업, html2canvas CDN 로드
├── style.css      # CSS Position 구조 선언, 가이드라인 애니메이션 정의
├── app.js         # 드래그 앤 드롭 마우스 좌표 계산 및 스타일 바인딩
└── README.md      # 본 문서 (프로젝트 안내서)
```

---

## 🚀 실행 방법

1. `meme-generator` 디렉토리 내의 **`index.html`** 파일을 더블클릭하여 사용하는 웹 브라우저에서 실행합니다.
2. 텍스트 입력 후 글자를 마우스로 잡고 캔버스 위에서 자유롭게 움직여보세요!
